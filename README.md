# myMiku

###日志

######2014-01-15

开坑

######2015-02-17

整理代码的时候又看到了这个好玩的项目。

出于怀念，再次重温了1年前的作品。

原计划做成一个养成miku的游戏，分为四个部分。最终只实现了节奏游戏的部分。

基于cocos2dx引擎。xcode6.1.1编译通过，iphone 4s(ios 8.1)运行成功
